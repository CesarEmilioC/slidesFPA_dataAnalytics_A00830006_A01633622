## Notes on the design
- Aspect ratio **16:9** for modern projectors.
- Tec de Monterrey colour palette (deep blue + warm gold), Open Sans typography.
- Margins fixed at 0.9 cm left / right; frame title sits in a coloured band so body content never bleeds off-slide.
- Footer carries authors, short title, and slide number — no navigation symbols.
- Title and thank-you slides are full-bleed with TikZ overlays; everything else uses the standard layout for readability.
